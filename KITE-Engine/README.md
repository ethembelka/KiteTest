# KITE-Engine
The core Engine, grid, test and dashboard agnostic.

# Command to Run
java -cp kite-jar-with-dependencies.jar:[Path and name of the test implementation].jar org.webrtc.kite.Engine [Path and name of config file].json

# Examples:
